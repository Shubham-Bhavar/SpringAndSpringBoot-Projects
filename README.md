# Spring & Spring Boot Projects 🚀

This repository contains multiple hands-on projects built using the Spring Framework.  
It covers different ways of configuring Spring applications and demonstrates core concepts of Dependency Injection.

---

## 📂 Projects Included

### 🔹 XML Configuration
- Project: spring1
- Demonstrates bean configuration using XML

### 🔹 Java-Based Configuration
- Project: spring-program2
- Uses @Configuration and @Bean annotations

### 🔹 Annotation-Based Configuration
- Project: spring-program3
- Uses @Component, @Autowired for Dependency Injection

---

## 🚀 Key Concepts Covered

- Dependency Injection (DI)
- Inversion of Control (IoC)
- Bean Lifecycle
- XML vs Java vs Annotation configuration

---

## 🛠️ Tech Stack

- Java
- Spring Framework (Core)
- Eclipse IDE
